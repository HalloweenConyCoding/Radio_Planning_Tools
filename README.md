# 📡 Radio Planning Tools

> **A mini telecom engineering tool suite for radio planning, antenna analysis, coverage visualization, and network data workflows.**

---

## 📡 About This Project

**Radio Planning Tools** is a collection of lightweight engineering utilities I created while working as a **Telecommunications Engineer specializing in Radio Planning**.

The project started as a personal working tool to reduce repetitive manual tasks, speed up analysis, and make technical planning results easier to visualize. Over time, it has grown into a more polished mini tool suite covering several practical areas of telecom engineering, including antenna behavior, signal coverage, path loss, tilt planning, RF parameter review, and data-driven network planning support.

This project is not intended to be a large commercial platform. It is a focused, practical, browser-based toolkit designed to help engineers, planners, analysts, students, or anyone interested in radio network work understand and perform common telecom calculations more clearly.

---

## 🎯 Project Purpose

The main goal of this project is simple:

> **Make radio-planning work faster, clearer, and easier to explain.**

In real telecom engineering work, many tasks involve repeated calculations, visual checks, parameter comparisons, or quick technical validation. These tasks are often small individually, but they consume time when repeated every day.

This tool suite was built to help with that problem by providing simple, visual, and accessible tools that can support workflows such as:

- antenna beam and radiation-pattern review
- RF coverage and planning calculation
- path loss and signal comparison
- tilt and coverage estimation
- telecom data checking and visualization
- engineering explanation and presentation support

The tools are designed to be practical first. The focus is not only on calculation, but also on making the results easier to inspect, compare, and communicate.

## 🛠️ What This Project Includes

This project may include multiple mini tools and pages, such as:

| Tool Area | Purpose |
|---|---|
| **Antenna / MSI Plotting** | Visualize antenna radiation patterns and compare beam behavior |
| **Tilt / Coverage Tools** | Estimate coverage behavior based on antenna height, tilt, and distance |
| **Power / FSPL Calculators** | Support quick RF power and path-loss related checks |
| **Planning Utilities** | Help with telecom planning, visualization, and engineering workflow support |
| **Main Landing Page** | A central navigation page for accessing the available tools |

Each tool is built to support a specific engineering task while keeping the interface lightweight and easy to use.

## 🌌 Design Direction

The current design direction uses a **dark RF command-console theme** inspired by telecom monitoring systems, antenna-radar interfaces, space-like signal environments, and technical engineering dashboards.

The visual style aims to feel:

- technical, but not overly complex
- modern, but still practical
- dark, clean, and readable
- suitable for engineering work
- more polished than a plain calculator page

The design uses elements such as dark panels, signal-inspired accents, subtle glow, structured cards, technical typography, and visual hierarchy to make the page feel like a proper engineering tool hub rather than a simple collection of links.

## 👷 Why I Built It

As a radio-planning engineer, many daily tasks involve combining technical calculation, parameter checking, data analysis, and visualization. Some tasks are too specific to be solved perfectly by general-purpose software, while building a full enterprise tool would be unnecessary.

This project was created to fill that gap.

It is a **small but useful engineering toolkit** built from real working needs. Instead of keeping it only as a private helper, I decided to publish it as a free resource so others in similar fields can use it, learn from it, or adapt the ideas for their own work.
