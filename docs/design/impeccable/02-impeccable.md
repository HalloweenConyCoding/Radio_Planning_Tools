# /impeccable

Category: Create  
Source: https://impeccable.style/docs/impeccable/  
Note: Paraphrased study note based on the public Impeccable docs, not a verbatim copy.

## Purpose

Use Impeccable’s full design intelligence when no specialized command clearly fits.

## Use when

- You are unsure which command to choose.
- The task spans layout, typography, color, motion, and product judgment.
- You want a broad design partner rather than a narrow pass.

## How it works

- Loads the opinionated design handbook and anti-pattern catalog.
- Reads PRODUCT.md and DESIGN.md when available.
- Uses project register: brand surfaces and product surfaces get different defaults.
- On first use, it may trigger the teach flow to create project context.

## Example prompt

```text
/impeccable redo this hero section
```

## Avoid / pitfalls

- Do not treat it as a simple style-guide linter.
- For refinement, prefer specific commands like polish, distill, critique, layout, or typeset.
- Run teach early so it does not need to ask context questions mid-task.

## Related commands

`/impeccable teach`, `/impeccable document`, `/impeccable craft`, `/impeccable shape`, `/impeccable live`
