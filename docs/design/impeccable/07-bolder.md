# /impeccable bolder

Category: Refine  
Source: https://impeccable.style/docs/bolder/  
Note: Paraphrased study note based on the public Impeccable docs, not a verbatim copy.

## Purpose

Push a safe, forgettable design toward stronger visual impact without chaos.

## Use when

- A landing page, hero, or content feature looks too generic.
- The design is technically fine but lacks presence.
- The project can handle a more assertive visual voice.

## How it works

- Amplifies scale, weight contrast, color commitment, and compositional confidence.
- Commits harder to existing choices instead of simply adding more visual elements.
- Uses techniques such as stronger display type, asymmetry, committed accents, and larger compositional moves.

## Example prompt

```text
/impeccable bolder the landing page hero
```

## Avoid / pitfalls

- Do not run it on dense operator tools, dashboards, or forms.
- Bold means committed, not loud.
- If it goes too far, follow with quieter.

## Related commands

`/impeccable quieter`, `/impeccable typeset`, `/impeccable colorize`, `/impeccable delight`
