# /impeccable quieter

Category: Refine  
Source: https://impeccable.style/docs/quieter/  
Note: Paraphrased study note based on the public Impeccable docs, not a verbatim copy.

## Purpose

Reduce visual intensity while preserving the design’s intent.

## Use when

- A UI is too loud, aggressive, or overstimulating.
- It has too many accents, gradients, shadows, animations, or decorative effects.
- A bolder pass went too far.

## How it works

- Reduces color intensity, contrast extremes, decoration, motion, and effects.
- Pulls accents back to one primary color plus muted support.
- Removes shadows, gradients, borders, blur, or parallax when they do not carry structure or meaning.
- Refines without neutralizing the design.

## Example prompt

```text
/impeccable quieter the pricing page
```

## Avoid / pitfalls

- Do not run it on designs that are already measured.
- Do not confuse quieter with distill; quieter reduces intensity, distill removes elements.
- If the problem is “too busy,” distill may be the better first move.

## Related commands

`/impeccable bolder`, `/impeccable distill`, `/impeccable polish`
