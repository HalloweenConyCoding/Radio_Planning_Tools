# /impeccable distill

Category: Simplify  
Source: https://impeccable.style/docs/distill/  
Note: Paraphrased study note based on the public Impeccable docs, not a verbatim copy.

## Purpose

Remove unnecessary UI complexity and reduce a design to its essential job.

## Use when

- A page feels cluttered, busy, noisy, or overloaded.
- Critique flags cognitive load or visual noise.
- A feature has grown by accretion and needs editing.

## How it works

- Starts by identifying the single job of the interface.
- Assesses sources of complexity: too many elements, excessive variation, overload, noise, confusing hierarchy, and feature creep.
- Removes, combines, hides, or consolidates elements so each remaining item earns its place.

## Example prompt

```text
/impeccable distill this dashboard
```

## Avoid / pitfalls

- Do not confuse distill with deleting needed features.
- Do not run it too early while the feature is still growing.
- If the right fix is arrangement rather than subtraction, use layout.

## Related commands

`/impeccable quieter`, `/impeccable polish`, `/impeccable layout`, `/impeccable critique`
