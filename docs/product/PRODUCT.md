# PRODUCT

## Name
Coverage Planning Tools

## Domain and Purpose
A local-first toolkit hub for radio planning engineers to access practical calculators and visualization tools quickly.

## Primary Users
- Radio planning / telecom engineers
- Technical users who need direct, fast utility access
- Returning users who compare current and legacy tool versions

## Product Goals
- Make the most useful tools easy to find and launch.
- Keep status clarity between stable, preview, and in-progress tools.
- Preserve trust with straightforward, engineering-first presentation.

## Personality
- Precise
- Practical
- Confident
- Calm

## Design Principles
- Utility first: scanning and navigation should be effortless.
- Strong hierarchy: clear sectioning and deliberate emphasis.
- Consistent systems: repeatable spacing, color, and component language.
- Respectful motion: subtle and meaningful, never distracting.

## Anti-References
- Generic AI-landing look with ornamental gradients and weak structure.
- Overly playful visual language that reduces technical trust.
- Heavy visual noise that competes with tool discovery.

## Accessibility Needs
- Maintain readable contrast across all card states.
- Keep clear focus-visible outlines for keyboard users.
- Preserve reduced-motion behavior where animations exist.
- Support small-screen scanning with stable spacing and type scale.
